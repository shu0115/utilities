# Utilities

## Usage

```
cd RAILS_ROOT
curl https://raw.github.com/shu0115/utilities/master/utility.rb > app/models/utility.rb
curl https://raw.github.com/shu0115/utilities/master/ja.yml > config/locales/ja.yml
curl https://raw.github.com/shu0115/utilities/master/translation_ja.yml > config/locales/translation_ja.yml
```
